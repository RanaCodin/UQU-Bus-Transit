# UQU Uni Transit System

UQU Uni Transit is a bus transportation management system designed for students at Umm Al-Qura University.  
The system organizes bus schedules, routes, and seat reservations, providing a clear and efficient transportation experience.

---

## Objective
Improve daily transportation by:
- Reducing waiting time and crowding
- Clarifying bus schedules and routes
- Allowing students to reserve seats
- Supporting real-time trip updates

---

## Key Features
- Secure account creation and login
- Upload class schedule to auto-align trip times
- Select home/pickup location
- View available bus routes and schedules
- Book or cancel bus rides
- Real-time bus tracking and arrival notifications
- Daily schedule adjustments
- Trip feedback and evaluation

---

## Core System Entities
- Student
- Bus
- Driver
- Route
- Schedule
- Reservation
- Payment
- Feedback
